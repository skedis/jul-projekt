# Jul-projekt

Simple Express server with Node.js.

## Requirements

* [Node.js & NPM](https://nodejs.org/en/)
* [Git](https://git-scm.com/)

## Installation
**Step 1**: Clone repository
```
git clone https://github.com/skedis/jul-projekt.git
```

**Step 2**: Install the required Node modules
```
npm install
```

## Usage
To run the app:
```
npm start
```
